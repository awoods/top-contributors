# Test Locally

1. Install the [Jekyll](https://jekyllrb.com/) gem
2. Install the `github-pages` gem. 
    * Here's a good guide: https://help.github.com/articles/setting-up-your-github-pages-site-locally-with-jekyll/
3. Clone this repo
4. Run `jekyll serve`
